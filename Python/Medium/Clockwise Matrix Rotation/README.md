# 🐍 Rotate Matrix by 90° Clockwise

## 📌 Problem Statement
You are given an **n × n 2D matrix** representing an image.

Your task is to **rotate the matrix by 90 degrees clockwise**.

> Equivalent to: `numpy.rot90(matrix, k=-1)`  
> (But without using any built-in functions or libraries)

---

## ⚠️ Bonus Challenge
Perform the rotation **in-place**:

- Modify the original matrix directly  
- **Do NOT** create another matrix
